# List of Marketing Popup Classes to Target

This document lists CSS classes, IDs, and selectors that can be used to hide marketing popups before taking screenshots.

## Summary of Common Selectors

These are the most common selectors found across templates:

```css
/* Webflow Badge */
.w-webflow-badge

/* Template Sales/Info Popups */
.master-sales-details
.mega-sales-info
.container-sales-details
.flex-cta-sales-info
.cta-sales-info
.overlay-sales-details
.link-template-details
.icon-template-details
```
